
# Reconnaissance de Pièces

Le but de projet est de déterminer à partir d'une image le nombre de pièces présentes sur cette dernière et calculer la valeur monétaire totale (en euros).

## Tech Stack

- Language : Python 3
- Librairies : NumPy, OpenCV
